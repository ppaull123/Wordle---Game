ABOUT WORDLE

About the Game

This is a wordle-inspired word guessing game built as a fullstack web application. The game lets you choose the word length (3-7 letters) and wether the word should have unique letters, then challenges you to guess the hidden word.

How to Play

1 Choose the number of letters (3-7) and wether you want unique letters.
2 Type a word and press enter to guess.
3 Each letter is colored based on feedback:
Green - Correct letter, correct position.
Yellow - Correct letter, wrong position.
Red - The letter is not in the word.

Technical Details

This project is built using React on the frontend and Node.js with Express on the backend. The game logic is handled on the server, where guesses are checked against the hidden word and feedback is generated for each letter. Words are loaded from a dictionary file, and winning scores are stored in MongoDB using Mongoose. The highscore page is rendered with Handlebars and supports filtering by word length and unique letters.
